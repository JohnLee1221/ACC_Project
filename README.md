 注意事项：

- carsim中的车辆质量：m = Sprung mass（簧上质量）+  Unsprung mass（悬架质量）*  2；
- carsim侧偏刚度，从Touring Tires中查找轮胎的垂向力对应的曲线，求斜率；
- carsim侧偏刚度的单位为：N/度，算法侧偏刚度单位为：N/弧度，单位需要换算；
- 算法侧偏刚度：carsim侧偏刚度*2 （算法为自行车模型）；
- 算法侧偏刚度为负值；
- Vehicle Body中把Aerodynamics选项关闭掉；
